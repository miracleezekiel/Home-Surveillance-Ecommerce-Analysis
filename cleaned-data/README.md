# Home Surveillance Ecommerce Analysis

## Project Overview

This project analyzes ecommerce data from the home surveillance and smart security industry to identify revenue drivers, inventory patterns, product performance trends, and seller activity. The analysis combines data validation, exploratory analysis, KPI development, business reporting, and SQL preparation to generate actionable business insights.

### Tools Used

* Google Sheets
* MySQL Workbench
* GitHub

### Skills Demonstrated

* Data Cleaning
* Data Validation
* Exploratory Data Analysis (EDA)
* KPI Development
* Revenue Analysis
* Inventory Analysis
* Business Intelligence
* SQL Fundamentals
* Business Reporting
* Data Documentation
