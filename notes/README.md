# Home Surveillance E-commerce Analysis

## Project Overview

This project analyzes a home surveillance e-commerce marketplace dataset containing surveillance cameras, DVR/NVR systems, smart security products, and related accessories.

The objective was to identify high-performing products, profitable categories, seller performance trends, pricing opportunities, and ecosystem adoption patterns.

## Business Questions

* Which products generate the highest estimated revenue?
* Which brands dominate the surveillance market?
* Which product categories perform best?
* How do discounts impact revenue generation?
* Which sellers demonstrate the strongest commercial performance?
* Which smart home ecosystems show the highest market presence?

## Tools Used

* Google Sheets
* Excel
* Pivot Tables
* Data Cleaning Techniques
* Feature Engineering
* GitHub
* MySQL Workbench

## Key Findings

* Premium surveillance products generated the highest estimated revenue.
* Security camera categories dominated marketplace activity.
* Ecosystem-integrated products demonstrated stronger commercial value.
* High seller reach correlated with increased revenue opportunities.
* Revenue performance varied significantly across pricing tiers.

## Recommendations

* Expand inventory in top-performing categories.
* Prioritize premium and ecosystem-compatible products.
* Monitor seller performance and global reach metrics.
* Use pricing tiers to optimize product positioning.
